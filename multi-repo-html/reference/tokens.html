Border Radius

Corner radius tokens. Values shown as mobile / desktop (corners step up one level at ≥768px).

--radius-none 0px Sharp — dividers, full-bleed banners
--radius-xs 4px / 8px Checkboxes, radio buttons, small tags
--radius-sm 8px / 12px Chips, compact cards, tooltips
--radius-md 12px / 16px Input fields, standard cards, dialogs
--radius-lg 16px / 24px Large cards, bottom sheets, modals
--radius-xl 24px / 32px Hero cards, contextual popovers
--radius-2xl 32px / 48px Extra-large hero surfaces
--radius-full 9999px Pills, FABs, badges, avatar containers
